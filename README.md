# 创世之掌 · 七日谈

> 用你的双手创造宇宙。
>
> *Create the cosmos with your bare hands.*

---

## 🖐 手势交互

| 手势 | 操作 |
|---|---|
| 🫴 手掌面向镜头推进 / 拉远 | 收缩星河，扩展宇宙 |
| 👋 手部转动 | 三百六十度转动寰宇 |
| 👌 拇指食指捏合保持 | 坠入微观之界 |
| ✊✊ 双手同时握拳 · 松开展开 | 翻越创世七日 |

桌面版键盘方向键 `←` `→` 也可前后切换七日。

---

## 📱 移动端触控

| 触控 | 操作 |
|---|---|
| ↔️ 左右滑动 | 切换创世七日 |
| 🤏 双指缩放 | 拉近拉远宇宙视界 |
| 👆 双击 | 踏入 / 离开微观世界 |

移动版页面同时支持摄像头手势识别——打开页面后点击 **📷 打开摄像头** 即可切换为手势控制。

---

## 🏠 本地运行

```bash
# 部署任意 HTTP 服务器即可
npx serve .
# 或
python3 -m http.server 8000
```

然后浏览器访问：
- 桌面版：`http://localhost:8000/index_v6.html`
- 移动版：`http://localhost:8000/index_mobile.html`

移动端若在平板本地运行，建议安装 **Simple HTTP Server** App，选择项目目录后启动，在浏览器访问 `http://127.0.0.1:8080/index_mobile.html` 即可体验摄像头手势。
